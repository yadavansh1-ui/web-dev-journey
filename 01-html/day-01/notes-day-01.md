# Day 1 - HTML Basics

## What I Learned

- HTML is used to create the basic structure of a website  
- HTML files have `.html` extension  
- Basic structure includes head and body  

## Boilerplate

- `! + Enter` in VS Code gives basic HTML structure  

## Tags

- Tags are written inside `< >`  
- Most tags have opening and closing form  

Example:  
`<p>This is a paragraph</p>`

## Attributes

- Attributes provide extra information inside tags
