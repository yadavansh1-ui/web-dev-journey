## What I Learned

---

## Types of CSS

### 1. Inline CSS
- used inside HTML tag  

Example:
```
<h1 style="color: red;">Hello</h1>
```

---

### 2. Internal CSS
- written inside `<style>` tag in `<head>`  

Example:
```
<style>
  h1 {
    color: blue;
  }
</style>
```

---

### 3. External CSS
- written in separate `.css` file  
- linked using `<link>`  

Example:
```
<link rel="stylesheet" href="style.css">
```

👉 `rel="stylesheet"` → tells browser it is CSS  
👉 `href` → path of CSS file  

---

## JavaScript in HTML

- added using `<script>` tag  

Example:
```
<script src="script.js"></script>
```

👉 `src` → path of JavaScript file  

---
