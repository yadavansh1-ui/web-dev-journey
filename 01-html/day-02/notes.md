# Day 2 - HTML Tags

## Headings

- HTML has 6 types of headings: <h1> to <h6>  
- <h1> is the largest and <h6> is the smallest  

Example:
<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>

---

## Paragraph

- <p> tag is used to write text in paragraph form  

Example:
<p>This is a paragraph explaining HTML basics.</p>

---

## Anchor Tag (Link)

- <a> tag is used to create links  
- href attribute defines the URL  

Example:
<a href="https://www.google.com">Go to Google</a>

---

## Image Tag

- <img> tag is used to display images  
- src attribute → image link  
- alt attribute → text if image doesn't load  

Example:
<img src="https://via.placeholder.com/150" alt="Sample Image">

---

## Unordered List

- <ul> is used for bullet list  
- <li> is used for list items  

Example:
<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

---

## Ordered List

- <ol> is used for numbered list  

Example:
<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>

---

## Attributes

- Attributes provide extra information to tags  
- Written inside opening tag  

Examples:
- href → used in <a> tag  
- src → used in <img> tag  
- alt → used in <img> tag
