# Day 4 - HTML Tables

## What I Learned

Today I learned how to create structured tables in HTML.

---

## Table Tags

- `<table>` → creates a table  
- `<tr>` → table row  
- `<th>` → table heading  
- `<td>` → table data  
- `<caption>` → table title  

---

## Basic Structure

```
<table>
  <tr>
    <th>Heading</th>
  </tr>
  <tr>
    <td>Data</td>
  </tr>
</table>
```

---

## Important Attributes

- `border` → adds border to table  
- `colspan` → merges columns  
- `rowspan` → merges rows  

---

## Colspan Example

```
<th colspan="2">Details</th>
```

---

## Rowspan Example

```
<th rowspan="2">Name</th>
```

---

## Caption

- `<caption>` → used to give title to table  

Example:
```
<caption>Student Information</caption>
```

---

## Why Tables?

- used to display structured data  
- used in reports, dashboards, forms  

---

Today I learned:
