# CSS Day 2 - Colors & Fonts

## What I Learned

Today I learned how to use different color formats and font styles in CSS.

---

## Types of Colors

### 1. Named Colors
```
color: red;
```

### 2. HEX Colors
```
color: #00ffcc;
```

### 3. RGB
```
color: rgb(255, 0, 0);
```

### 4. RGBA (with transparency)
```
color: rgba(255, 0, 0, 0.5);
```

### 5. HSL
```
color: hsl(200, 100%, 50%);
```

---

## Font Properties

- font-size → size of text  
- font-weight → boldness  
- font-style → italic  
- font-family → type of font  

Example:
```
font-family: Arial, sans-serif;
```

---

## Key Points

- HEX and RGB are most used  
- RGBA is used for transparency  
- font-family should always have backup fonts
